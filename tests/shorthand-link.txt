Shorthand link Span

------------------------------------------------

-- div

>>alfa>>

------------------------------------------------

<div class="basic-full-div">
    <p><a href="">alfa</a></p>
</div>


################################################

Shorthand link Span With Key Value

------------------------------------------------

-- div

>>alfa|key1: value1>>

------------------------------------------------

<div class="basic-full-div">
    <p><a href="" key1="value1">alfa</a></p>
</div>

################################################

Shorthand link Span With Multiple Key Value

------------------------------------------------

-- div

>>alfa|key1: value1|key2: value2>>

------------------------------------------------

<div class="basic-full-div">
    <p><a href="" key1="value1" key2="value2">alfa</a></p>
</div>


################################################

Shorthand link Span With Flag

------------------------------------------------

-- div

>>alfa|flag1>>

------------------------------------------------

<div class="basic-full-div">
